Conteudo adicional

Voce ai, sim voce mesmo, ficou com vontade de fazer o seu robo? nesta pagina sera citados alguns tutoriais e dicas.
Eh importante saber que, existem varios jeitos de se fazer um, tais como: virtuais com machile learning ou deep learning, varias linguagens de programacao disponiveis, lego, material pronto como o usado aqui e entre outros.
Vamos resumir aqui tais topicos mas eh aconselhavel que voces pesquisem para se aprofundar no assunto.

Tipos de robos

Jeitos de fazer seu proprio robo.

Machine learning

Ou aprendizado de maquina, eh algo que vem ganhando atencao nos ultimos anos, se baseia em treinar um "cerebro vazio" usando dados coletados, geralmente feito em python.

Deep learning

Ou as famosas "IAs", um assunto que "explodiu" dado a quantia de chats sendo criados, se baseia em treinar uma rede neural artificial, geralmente feito em python tambem, porem eh possivel fazer em C#, Java e entre outras.

Recomendacao de linguagem

Como citado a esquerda, varias linguagens de programacao fazem o mesmo servico, nos aconselhamos que voce procure por si so e escolha a que mais se encaixa no seu gosto.

Robos "fisicos"

Nao da para falar de robos sem citar arduino, o arduino serve para facilitar o aprendizado de programacao usando robos, tendo inclusive kits de arduino que sao recomendados.

Com arduino

Arduino vem com pecas prontas, voce monta e programa, dado a facilidade + a quantia de tutoriais na internete, eh recomendado para iniciantes.

Com lego

Existem pecas especificas de lego, inclusive com uso de pneumatica, para fazer robos, nao eh recomendado para iniciantes.
